# Fly_Fish Server 官方网站

Fly_Fish Server（称号-月半鱼的MC个人服务器）官方网站，一个基于 HTML5 UP Forty 模板构建的 Minecraft 多人游戏服务器展示站点。

## 项目简介

本网站是 Fly_Fish Server 的官方信息门户，为玩家提供服务器介绍、成员展示、图库浏览等功能。服务器自 **2022年12月25日** 起运营，由服主 **newbie_Ffish** 创建和维护。

## 服务器类型

网站涵盖以下 Minecraft 服务器模式：

- **原版生存 (Vanilla Survival)** — 经典原版 Minecraft 生存体验
- **枪战联机 (Gunfight Online)** — 枪战对抗玩法
- **持枪旅游 (Travel with Guns)** — 带枪械的探索旅游模式

## 网站页面结构

| 页面 | 说明 |
|------|------|
| `index.html` | 首页，展示服务器运行计时及导航入口 |
| `servers.html` | 服务器玩法介绍（原版生存/枪战联机/持枪旅游） |
| `team.html` | 玩家与团队成员展示 |
| `Vanilla.html` | 原版生存服务器详情 |
| `GunFight.html` | 枪战联机服务器详情 |
| `wait.html` | 服务器图库 & 物理硬件（建设中） |
| `player.html` | 玩家相关页面 |
| `404.html` / `4041.html` | 404 错误页面 |

## 技术栈

- **前端框架**：HTML5 UP Forty 模板
- **样式预处理**：Sass (SCSS)
- **JavaScript 库**：jQuery, Scrollex, Scrolly
- **图标字体**：Font Awesome 5
- **在线客服**：V5KF
